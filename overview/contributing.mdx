## Setup your development environment

Clone the repository:

```git
git clone https://github.com/Surfer-Protocol/Protocol.git
```

Navigate to the project directory:

```javascript
cd Protocol
```

Install the Mintlify CLI globally:

```javascript
npm install -g mintlify
```

Run the development server:
    
```javascript
mintlify dev
```

The website should now be running on [http://localhost:3000](http://localhost:3000).

## Add your platform

To add your platform, please follow the instructions in the [Adding Platforms](/platforms/adding/platforms) guide.

## Adding exporting methods for a platform

To add methods for exporting data from a platform, please follow the instructions in the [Adding Methods](/platforms/adding/methods) guide.

## Deploy your changes

Submit a PR to the `main` branch and we will review and merge it.
